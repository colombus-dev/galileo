# Galileo

A rich and interactive visualization interface for effective reporting on Machine Learning code exploration

## Final report (in Git)

The final report structure (Markdown pages + appendices + references) is in:

- [docs/README.md](docs/README.md)

## Technical Stack

React + TypeScript + Vite

## Installation with npm

Installation nodes module:
```bash
$ npm install
```

## Installation whith yarn
```bash
$ yarn install
```

## Launch software whit npm

Launch software:
```bash
$ npm run dev
```

Launch Storybook (documentations):
```bash
$ npm run storybook
```

## Launch software whit yarn

Launch software:
or
```bash
$ yarn dev
```

Launch Storybook (documentation):
```bash
$ yarn storybook
```

## Deployment

Using docker compose (recommended):

```bash
$ docker compose build
$ docker compose up
```
